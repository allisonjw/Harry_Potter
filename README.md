# Harry Potter Game
## By [Allison Wagner](https://github.com/allisonjw)

## Overview
Harry Potter Game is a web based application that allows Wizards and Witches to on view characters based on their houses! Additionally you will be able to select favorite characters and view your favorite collection. You can also view all spells.

## Getting started
* Clone down the repo and install the dependencies with `npm install`.
* Run `npm start` and visit http://localhost:3000 to view the application in the browser.
* Run the test suite with `npm test`.
